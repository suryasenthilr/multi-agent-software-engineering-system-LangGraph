# Simple Expense Calculator

A lightweight web application that lets users quickly add, delete, and reset expense entries while automatically calculating and displaying the total amount spent.

## Tech Stack
- **HTML** – Structure of the app (`index.html`).
- **CSS** – Styling and responsive layout (`styles.css`).
- **JavaScript** – Interactive functionality (`script.js`).

## Features
- Add an expense with a description and amount.
- Delete individual expense items.
- Reset all expenses at once.
- Real‑time calculation of the total expense.
- Responsive design that works on desktop and mobile devices.

## Setup
1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```
2. Navigate into the project folder:
   ```bash
   cd <project-folder>
   ```
3. Open **`index.html`** in any modern web browser. No additional build steps or dependencies are required.

## Usage
1. **Add an expense** – Fill in the *Description* and *Amount* fields, then click the **Add** button. The new entry appears in the list and the total updates automatically.
2. **Delete an expense** – Click the **Delete** (✖) button next to the expense you want to remove. The total recalculates instantly.
3. **Reset all expenses** – Press the **Reset** button to clear the entire list and set the total back to zero.
4. **View total** – The running total is displayed at the bottom of the expense list.

## Responsive Design
The layout adapts to various screen sizes. On mobile devices, the form fields and buttons stack vertically for easy tapping, and the expense list remains scrollable.
