# Multi-Agent Software Engineering System

**Multi-Agent Software Engineering System** is an AI-powered application generator built with LangGraph.

It simulates a software engineering workflow using specialized AI agents that collaborate to transform a natural language prompt into a complete application. From project planning and task decomposition to code generation, each agent is responsible for a specific stage of the development lifecycle, enabling automated software creation through a structured multi-agent pipeline.

---

## Architecture

* **Planner Agent** – Analyzes the user request and creates a structured project plan.
* **Architect Agent** – Breaks the plan into detailed implementation tasks with dependencies and integration details.
* **Coder Agent** – Implements each task, generates code, and writes files using available development tools.

## Getting Started

### Prerequisites

* Python 3.10+
* A Groq API Key

### Installation

Clone the repository:

```bash
git clone <repository-url>
cd multi-agent-software-engineering-system
```

Create and activate a virtual environment:

**Windows**

```bash
python -m venv .venv
.venv\Scripts\activate
```

**Linux/macOS**

```bash
python -m venv .venv
source .venv/bin/activate
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Create a `.env` file:

```env
GROQ_API_KEY=your_api_key_here
```

### Run

```bash
python main.py
```

Enter a project prompt when prompted by the CLI.
